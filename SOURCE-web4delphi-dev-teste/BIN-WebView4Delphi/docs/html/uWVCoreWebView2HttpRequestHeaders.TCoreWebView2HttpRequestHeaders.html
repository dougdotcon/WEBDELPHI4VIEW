Declaration

type TCoreWebView2HttpRequestHeaders = class(TObject)

Description

HTTP request headers. Used to inspect the HTTP request on WebResourceRequested event and NavigationStarting event.

It is possible to modify the HTTP request from a WebResourceRequested event, but not from a NavigationStarting event.

<see href="https://learn.microsoft.com/en-us/microsoft-edge/webview2/reference/win32/icorewebview2httprequestheaders">See the ICoreWebView2HttpRequestHeaders article.)
