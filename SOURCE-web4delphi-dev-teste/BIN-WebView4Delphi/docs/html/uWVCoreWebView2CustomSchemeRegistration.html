Class TCoreWebView2CustomSchemeRegistration

Represents the registration of a custom scheme with the CoreWebView2Environment.

This allows the WebView2 app to be able to handle WebResourceRequested event for requests with the specified scheme and be able to navigate the WebView2 to the custom scheme.
