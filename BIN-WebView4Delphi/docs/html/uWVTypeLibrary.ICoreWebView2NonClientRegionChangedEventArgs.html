function Get_RegionKind(out value: COREWEBVIEW2_NON_CLIENT_REGION_KIND): HResult; stdcall;

This property represents the COREWEBVIEW2_NON_CLIENT_REGION_KIND which the region changed event corresponds to. With this property an app can query for a collection of rects which have that region kind by using QueryNonClientRegion on the composition controller.
