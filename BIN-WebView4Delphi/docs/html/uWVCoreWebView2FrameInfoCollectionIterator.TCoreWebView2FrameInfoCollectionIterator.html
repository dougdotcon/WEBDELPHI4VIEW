property HasCurrent : boolean read GetHasCurrent;

`TRUE` when the iterator has not run out of `FrameInfo`s. If the collection over which the iterator is iterating is empty or if the iterator has gone past the end of the collection, then this is `FALSE`.

<see href="https://learn.microsoft.com/en-us/microsoft-edge/webview2/reference/win32/icorewebview2frameinfocollectioniterator#get_hascurrent">See the ICoreWebView2FrameInfoCollectionIterator article.)
