Description

This is a continuation of the ICoreWebView2DevToolsProtocolEventReceivedEventArgs interface that provides the session ID of the target where the event originates from.

function Get_sessionId(out value: PWideChar): HResult; stdcall;

The sessionId of the target where the event originates from. Empty string is returned as sessionId if the event comes from the default session for the top page.
