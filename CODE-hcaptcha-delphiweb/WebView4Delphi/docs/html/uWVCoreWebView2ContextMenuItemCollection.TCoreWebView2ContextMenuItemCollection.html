type TCoreWebView2ContextMenuItemCollection = class(TObject)

Description

Represents a collection of ContextMenuItem objects. Used to get, remove and add ContextMenuItem objects at the specified index.

<see href="https://learn.microsoft.com/en-us/microsoft-edge/webview2/reference/win32/icorewebview2contextmenuitemcollection">See the ICoreWebView2ContextMenuItemCollection article.)
