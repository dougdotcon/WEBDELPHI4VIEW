function GetContent(const aHandler : ICoreWebView2WebResourceResponseViewGetContentCompletedHandler) : boolean;

Get the response content asynchronously. The handler will receive the response content stream.

This method returns null if content size is more than 123MB or for navigations that become downloads or if response is downloadable content type (e.g., application/octet-stream). See `add_DownloadStarting` event to handle the response.

If this method is being called again before a first call has completed, the handler will be invoked at the same time the handlers from prior calls are invoked.

If this method is being called after a first call has completed, the handler will be invoked immediately.

TCoreWebView2WebResourceResponseView.GetContent will trigger the TWVBrowserBase.OnWebResourceResponseViewGetContentCompleted event with the resource contents.

<see href="https://learn.microsoft.com/en-us/microsoft-edge/webview2/reference/win32/icorewebview2webresourceresponseview#getcontent">See the ICoreWebView2WebResourceResponseView article.)
