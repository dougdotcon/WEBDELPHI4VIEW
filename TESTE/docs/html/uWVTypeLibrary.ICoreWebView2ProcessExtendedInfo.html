function Get_AssociatedFrameInfos(out frames: ICoreWebView2FrameInfoCollection): HResult; stdcall;

The collection of associated `FrameInfo`s which are actively running (showing or hiding UI elements) in this renderer process. `AssociatedFrameInfos` will only be populated when this `CoreWebView2ProcessExtendedInfo` corresponds to a renderer process. Non-renderer processes will always have an empty `AssociatedFrameInfos`. The `AssociatedFrameInfos` may also be empty for renderer processes that have no active frames.
